# Hirely – Interactive Job Portal with ATS

Hirely is a full‑stack **interactive online job portal** with an integrated **application tracking system (ATS)** built on the MERN stack.

## Tech Stack (Planned)

- Frontend: React, TypeScript, React Router, Redux Toolkit, Tailwind/MUI
- Backend: Node.js, Express.js, MongoDB (Mongoose), JWT auth
- Realtime: Socket.io
- DevOps: Docker, GitHub Actions, MongoDB Atlas, cloud deployment

## High‑Level Features

- Role‑based access: Job seeker, recruiter, admin, employer
- Job search with advanced filters and saved jobs
- Application submission and status tracking
- Recruiter dashboard with pipelines and analytics
- Real‑time notifications (status changes, new applications)
- Admin panel for users, jobs and platform analytics
